# skins-marketplace-2.0
Я хочу повторно написать проект на Spring frameworks, связанный с сайтом продажи скинов. Я уже делал этот проект по этой идее, но он был ближе к семестровой работе для предмета в институте. В этом проекте я хочу начать всё с чистого листа, применив накопленные мной знания и опыт, который я приобрел во время прохождения курса от Т-Академии.

Цель проекта:
1) Улучшить аналитическое мышление
2) По сравнению с первой версией проекта(семестровая работа), хочу добавить тестирование, работу с брокерами сообщениями, кэширование и т.д
3) Изучить новые подходы разработки. Например, нужно куда строже сделать перевод денег из одного баланса на другой при покупке товаров
4) Укрепить уже имеющиеся знания

Дополнительные цели:
1) Понять базовый js для ajax-запросов
2) Немного погрузиться во frontend. Прям немного :D

План:
Я не собираюсь сразу писать код.
Сначала построю архитектуру проекта, endpoint-ы, какие сущности нужны и как они буду взаимодействовать друг с другом, какие сценарии тестирования нужно сделать. Необходимо будет также сделать схему базы данных.
После составления плана приступлю к написанию года.

Планы большие, но нужно хотя бы довести состояние прокета до Minimum Viable Product.
