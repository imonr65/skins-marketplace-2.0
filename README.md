# skins-marketplace-2.0

Я хочу повторно написать проект на Spring frameworks, связанный с сайтом продажи скинов. Я уже делал этот проект по этой идее, но он был ближе к семестровой работе для предмета в институте. В этом проекте я хочу начать всё с чистого листа, применив накопленные мной знания и опыт, которые я приобрел во время прохождения курса от Т-Академии.

Цель проекта:
1) Улучшить аналитическое мышление
2) По сравнению с первой версией проекта(семестровая работа), хочу добавить тестирование, работу с брокерами сообщениями, кэширование и перейти на микросервисную архитектуру
3) Изучить новые подходы разработки
4) Укрепить уже имеющиеся знания

Дополнительные цели:
1) Не знаю получится ли, но хочу попробовать написать openApi-контракт
2) Понять базовый js для ajax-запросов
3) Немного погрузиться во frontend. Прям немного :D

План:
Я не собираюсь сразу писать код.
Сначала попробую построить архитектуру проекта, endpoint-ы, какие сущности нужны и как они буду взаимодействовать друг с другом, какие сценарии тестирования нужно сделать. Необходимо будет также сделать схему базы данных.
После составления плана приступлю к написанию года.

Планы большие, но нужно хотя бы довести состояние прокета до Minimum Viable Product.

Удачи мне на моем пути 'разработчика'.
